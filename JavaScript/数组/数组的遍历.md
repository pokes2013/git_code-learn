### 数组的遍历

#### 使用for循环遍历数组

```javascript
      let arr = ["red", "green", "blue"];
      for (let item = 0; item < arr.length; item++) {
          document.write(arr[item]+"<br />");
      }
```

![1725454142873](数组的遍历~image/1725454142873.png)

结果：

red
green
blue 

#### 使用for...in遍历数组

```javascript
      let arr2 = ["red2", "green2", "blue2"];
      for (let item in arr2) {
        document.write(arr2[item] + "<br />");
      }
```

![1725457484431](数组的遍历~image/1725457484431.png)

结果：

red2
green2
blue2
